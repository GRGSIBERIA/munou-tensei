# 無能転生 要件定義（ドラフト）

## 1. 目的
- プレイヤーが「無能」な主人公を育成し、試行錯誤を通じて成長を実感できる体験を提供する。
- 1プレイ 20〜40分程度で、短時間でも進捗が感じられるゲームサイクルを実現する。

## 2. 想定プラットフォーム
- PC（Windows/Mac）を初期対象とする。
- 将来的なモバイル展開を考慮し、入力・UIは拡張可能な設計にする。

## 3. ターゲットユーザー
- 成長・育成系ゲームが好きな15〜35歳。
- 高難度よりも「失敗からの改善」が楽しいユーザー。

## 4. コアゲームループ
1. クエスト/行動を選択する。
2. 成功/失敗に応じてステータス・資源が変動する。
3. 獲得した資源で育成・装備更新を行う。
4. 次の挑戦でより高い成果を狙う。

## 5. 機能要件（MVP）
- キャラクター基本ステータス（例: 体力、知力、器用さ）。
- 行動選択UI（最低3種類の行動）。
- 成否判定ロジック（確率 + ステータス補正）。
- 報酬システム（経験値、通貨、アイテムのいずれか）。
- セーブ/ロード（ローカル保存）。
- チュートリアル（初回プレイ3分以内で基本操作を理解可能）。

## 6. 非機能要件（MVP）
- 起動時間: 10秒以内（ターゲットPC環境）。
- 平均フレームレート: 60fps（1080p、標準設定）。
- クラッシュ率: 1%未満（内部テスト）。
- 主要操作はキーボード/ゲームパッド双方で実行可能。

## 7. 受入基準（初期）
- 新規プレイヤーが10分以内に「1回の成長サイクル」を完了できる。
- 連続5回の行動結果に対して、成否判定が仕様どおり再現される。
- セーブ後に再起動しても、ステータスと進行状況が復元される。

## 8. 今後の検討項目
- 世界観・シナリオの具体化。
- 育成ビルドの多様性（スキルツリー等）。
- リプレイ性向上のためのランダムイベント設計。
