# 経済システム要件

## 貨幣体系
- 貨幣は鋳貨を基本とする。
- 硬貨はランクが低い順に以下の9種類。
  1. 小銅貨
  2. 中銅貨
  3. 大銅貨
  4. 小銀貨
  5. 中銀貨
  6. 大銀貨
  7. 小金貨
  8. 中金貨
  9. 大金貨
- 10枚ごとに1つ上の硬貨へ換金できる。

## 生活コスト
- 大人は1人あたり1日「7小銅貨」で生活可能。
- 生徒の家賃は1日「5小銅貨」。

## 生徒会の赤字ペナルティ
- 生徒会の予算がマイナスの状態で1カ月経過した場合、領主が生徒会追い出しの圧力をかける。
- 赤字解消のため、領主・教皇・商工会が生徒を強制的に雇用する。
- 2カ月連続で赤字が続いた場合、領主が生徒を奴隷として売却する。
- 3カ月連続で赤字が続いた場合、経営破綻として生徒会が追い出され、ゲームオーバーとなる。
