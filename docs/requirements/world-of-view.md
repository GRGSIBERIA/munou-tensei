# 無能転生 世界観ドラフト

## 1. 世界設定の概要
- 舞台は、なろう系の中世ヨーロッパ風異世界。
- 現代日本から転生した「平凡な生徒」100人が、この世界で生活基盤を築きながら生き抜く。
- プレイヤーは集団の一員として、学園運営・生活・政治的立ち回りを通じて生徒たちの未来を左右する。

## 2. 転生した生徒たち
- 生徒は全員「突出した才能を持たない」状態から始まる。
- それぞれに適性や性格傾向があり、職業選択や所属先の影響で成長方向が変化する。
- 生徒は原則として「生徒会」への参加が義務付けられる。

## 3. 四大派閥

### 3.1 生徒会
- 100人の生徒全員が所属する基盤組織。
- 日常生活の維持（食料、住居、治安、教育）と、外部勢力との調整窓口を担う。
- 実質的には自治政府として機能し、意思決定がゲーム進行の中心となる。

### 3.2 領主派
- 土地・兵力・税制を掌握する封建貴族勢力。
- 現実的な秩序と実利を重視し、農業・軍事・インフラ分野の職業に強い影響を与える。
- 領主派との関係が深いほど、安全保障や資源供給は安定しやすい一方で、自治の自由度が制限される。

### 3.3 教皇派
- 宗教的権威と教会ネットワークを持つ勢力。
- 医療・福祉・教育・道徳規範の分野で影響力が大きい。
- 教皇派との関係が深いほど、民衆支持や精神的安定を得やすい一方で、教義による行動制約が増える。

### 3.4 商工会
- 多様な職業を束ねるギルド組織で、街の商業活動を管理する勢力。
- 商業・流通・生産に関わる職業の登録と人材管理を行う。
- 多くの職種は生徒受け入れに消極的だが、稀に重労働枠（鉱山労働、農業、林業など）の募集が発生する。
- 重労働系は高収入になりやすい一方で、事故・疾病・治安リスクが高い。

## 4. 派閥関係の基本構造
- 生徒は全員、生徒会への参加を強制される。
- ただし、就く職業によって領主派・教皇派・商工会から受ける影響度が変化する。
- 例:
  - 騎士・衛兵・開拓官は領主派の影響を受けやすい。
  - 司祭補佐・治療師・書記官は教皇派の影響を受けやすい。
  - 鉱夫・林業従事者・季節農業労働者は商工会の影響を受けやすい。
- 生徒会は中立組織ではなく、各派閥の圧力を受けながら集団全体の最適解を探る立場となる。

## 5. プレイ体験における世界観の役割
- プレイヤーは「個人の成功」よりも「100人の共同体の存続と発展」を優先して判断する。
- 同じ出来事でも、領主派・教皇派・商工会・生徒会のどの視点を重視するかで結果が変わる。
- 高収入だが危険な重労働を誰に割り当てるかが、倫理・収支・安全保障のジレンマを生む。
- 世界観そのものが、職業選択・資源配分・対外交渉・内部統治のジレンマを生み出す設計とする。

## 6. 今後の詳細化ポイント
- 100人の初期属性（性格、適性、関係性）の定義。
- 職業カテゴリごとの派閥影響パラメータ設計。
- 領主派・教皇派・商工会との友好/対立がイベント分岐へ与える影響量の数値化。
- 重労働職のリスク（負傷率、死亡率、離脱率）と報酬バランスの設計。
- 生徒会内部の意思決定方式（多数決、役職制、信任制など）のルール化。
